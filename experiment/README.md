# AFrameXR User Study

Thank you for participating!

## Introduction

The goal of this experiment is to evaluate the usability and ease of use of AFrameXR for creating interactive 3D/XR
visualizations.

The experiment is divided into two notebooks:

1. **Ease of use Notebook** — creating 3D visualizations in the browser.  
   **Estimated duration:** ~10 minutes

2. **Usability Notebook** — immersive interaction in Desktop, Virtual Reality (VR) and Augmented Reality (AR).  
   **Estimated duration:** ~5 minutes

**Total estimated duration:** ~15 minutes

## Before Starting

- The notebooks may ask you to install dependencies the first time you run them; please accept and proceed with the
installation if prompted. These dependencies are downloaded only to run the experiment inside the browser-based (WASM)
environment. They are temporary, are removed when you close the session, and do not modify your computer or install
anything system-wide. In this context, this is a normal step required to run the notebook correctly.
- In marimo, use the **"Run All"** button located in the bottom-right corner after opening the notebook (it might last
a couple of minutes, depending on the computer).
- Each notebook contains guided tasks.
- After completing each notebook, you will be asked to answer a short questionnaire about your experience.
