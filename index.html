BabiaXR Python: 3D visualization in Python

BabiaXR Python is a 3D visualization library in Python using Marimo notebooks. With very similar sintaxis to Vega-Altair library and Vega-lite grammar, so you can import your graphics and display them in 3D.
import babiaxr
